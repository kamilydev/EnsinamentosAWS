☁️ Desafio AWS CloudFormation

Esse projeto faz parte do desafio da DIO.
Usei o CloudFormation para criar uma infraestrutura automatizada na AWS, começando com um bucket S3.

O que fiz

Criei um template YAML para a Stack

Subi a Stack no CloudFormation

Acompanhei a criação e verifiquei os recursos

Ferramentas e conceitos de automação que explorei

CloudFormation: infraestrutura como código (IaC)

Lambda: executar código sem gerenciar servidores

Step Functions: automatizar fluxos entre serviços

CodePipeline: automatizar ciclo CI/CD

Systems Manager: automatizar tarefas operacionais

CLI/SDKs: scripts para gerenciar recursos AWS
